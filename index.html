TrustCode

Shared trust, refreshed every 30 seconds.

TrustCode is a local TOTP secret generator for authenticator apps, family anti-scam checks, private shared verification, and developer testing.

Local-first
No account
QR export
Authenticator compatible
Python version
Run locally.

Install dependencies with pip install -r requirements.txt, then run python trustcode.py.

Use case

Share one secret with trusted family members. When someone claims to be you in an urgent situation, compare the current code before taking action.

Flow
1

Generate a local secret with the Python tool.

2

Import it into a TOTP-compatible authenticator app.

3

Use the changing code as an extra trust check with people you already trust.

Security note

TrustCode is a helper tool. It does not replace official identity verification, banking security, legal checks, or emergency services.
